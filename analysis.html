⚡ Energiabilanss
ALUS
Söödud vs põletatud kalorid päevas. Rohelised tulbad = söödud, oranžid = Garmin põletatud (BMR + aktiivsus). Sinine joon = bilanss (defitsiit / ülejääk).
😴 Uni → järgmise päeva söögiisu
LEPTIIN · GRELIIN
Kehv uni langetab leptiini (küllastustunne↓) ja tõstab greliini (nälg↑). Meta-analüüsid: lühike uni → ca +300–500 kcal järgmisel päeval (Hogenkamp et al., 2013; Spiegel et al., 2004). Siin: eelmise öö une skoor (sinine joon) vs järgmise päeva söödud kalorid (rohelised tulbad).
🥩 Toitumine → une kvaliteet
TRÜPTOFAAN · KIUDAINED
Valk sisaldab trüptofaani → serotoniin → melatoniin → kiirem uinumine. Kõrge kiudainete tarbimine korreleerub rohkema sügava unega (Grandner et al., 2015). Siin: päeva valk (oranž) + kiudained (roheline) vs sama öö une skoor (sinine joon).
💪 Toitumine → treeningvalmidus
GLÜKOGEEN · HRV
Garmin Training Readiness skoor põhineb HRV-l, unel ja treeningkoormusel. Süsivesikud taastavad glükogeeni (~1g/kg KK pärast trenni), valk toetab lihaste taastumist. Siin: eelmise päeva valk (oranž) + süsivesikud (sinine) vs järgmise päeva readiness skoor (roheline joon).
❤️ Kaloridefitsiit vs puhkepulss
METABOOLNE STRESS
Suur defitsiit (>700 kcal) tõstab kortisooliitaset → puhkepulss ↑, HRV ↓. Punane = defitsiit, roheline = ülejääk. Punane joon = järgmise hommiku puhkepulss.
💧 Hüdratatsioon vs stressitase
DEHÜDRATATSIOON
Isegi 1–2% kehakaalust veepuudus → tähelepanu↓, stressimarkerid↑, Garmin stressiskoor↑ (Armstrong et al., 2012). Siin: joodud vesi ml (sinised tulbad) vs Garmin päevane stressiskoor (punane joon, madalam = parem).
